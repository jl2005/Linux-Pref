# CPU 测试环境搭建

## 安装手册

Centos 7 最小安装中没有相关的手册，需要使用如下命令进行安装

```
yum install -y man-pages
```

## 增加虚拟机的磁盘

可以按照如下的步骤给虚拟机增加磁盘：

```
设置 -> 存储 -> SATA控制器->右击，选择“添加虚拟硬盘”
```

然后在linux中可以使用如下命令查看新增加的磁盘：

```
sudo fdisk -l
```

## 开启swap

```
# 创建Swap文件
$ fallocate -l 8G /mnt/swapfile
# 修改权限只有根用户可以访问
$ chmod 600 /mnt/swapfile
# 配置Swap文件
$ mkswap /mnt/swapfile
# 开启Swap
$ swapon /mnt/swapfile
```

## 关闭Swap

```
swapoff -a
```
